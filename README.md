# Movie Rating Database Management System
### Overview
This project demonstrates how to design, build, and query a structured movie rating database using SQL.
Core records managed by the database include:
•	Movies available in the system
•	Registered users who rate movies
•	Movie ratings provided by users
## Database Structure
The database maintains three core tables:
### Movies
Stores information about movies available in the system.
Fields include:
•	Movie ID
•	Title
•	Release Year
•	Genre
•	Director
### Users
Stores information about registered users who rate movies.
Fields include:
•	User ID
•	Name
•	Age
### Ratings
Tracks which users rated which movies.
Fields include:
•	Rating ID
•	User ID
•	Movie ID
•	Rating Value
•	Rating Date
## Features Demonstrated
This project demonstrates several SQL concepts:
•	Database creation
•	Table creation with constraints
•	Primary and foreign keys
•	Filtering with WHERE
•	Logical operators
•	Sorting with ORDER BY
•	Using DISTINCT, and LIMIT for pagination
## Written Queries
Queries for extracting and filtering data
Queries to demonstrate the use of comparison and logical operators for analytical insights
 Queries to organize, paginate, and refine query results
